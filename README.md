# CV Parser

A comprehensive system for parsing, analyzing, and extracting information from Curriculum Vitaes (CVs) / Resumes.

## Project Structure

- `frontend/`: Web user interface.
- `backend/`: Application backend API.
- `ai-service/`: Machine learning / AI service for parsing and text extraction.
- `datasets/`: Raw and processed dataset files for training and evaluation.
- `shared/`: Shared utilities, types, and configurations.
- `deployment/`: Configuration files for deployments (Kubernetes, Terraform, etc.).
- `docs/`: Design documents, API reference, and manuals.

## Getting Started

### Prerequisites

- Docker and Docker Compose
- Python 3.10+
- Node.js 18+

### Running the Project

You can run the entire application using Docker Compose:

```bash
docker compose up --build
```
